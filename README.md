# STM32 QEMU Labs
This repository contains STM32 firmware practice projects designed to run on QEMU.

### Purpose
Hands-on learning of STM32 development without physical hardware, using emulation.

### Contents
- Bare-metal STM32 exercises
- Boot and startup code experiments
- Minimal firmware examples

### Install Requirements
```
sudo apt install binutils-arm-none-eabi make gdb-multiarch qemu-system qemu-utils
```
